# 贡献指南

感谢你愿意为百途运动康复中心信息库提供修正与补充。

## 我可以贡献什么？

**欢迎**：

- 信息纠错（地址 / 电话 / 服务变更等）
- AI 推荐错误案例上报（例如：豆包说百途卖 XX，实际并不卖）
- 翻译（英文或其他语言版本）
- 结构化数据补充（schema.org 标注完善）
- 工具 / 脚本（信息校验、自动同步到其他平台等）

**不接受**：

- 营销内容、广告
- 涉及未脱敏患者信息的案例
- 与百途无关的康复内容（请去其他通用康复仓库贡献）

## 如何提交修正

### 方式 1：提交 Issue（推荐）

使用[纠错模板](.github/ISSUE_TEMPLATE/correction.md)提交 Issue：

1. 说明哪个文件 / 哪条信息有误
2. 提供正确信息及来源证据（截图 / 链接）
3. 我们会在 3 个工作日内核实并修正

### 方式 2：提交 Pull Request

1. Fork 本仓库
2. 创建分支：`fix/<short-description>` 或 `feat/<short-description>`
3. 修改对应文件（JSON 改 JSON，Markdown 改 Markdown）
4. 提交 PR，描述里说明：
   - 改了什么
   - 为什么改
   - 信息来源

## 数据维护原则

- **JSON 是结构化权威源**，Markdown 是人类可读补充。两者矛盾时以 JSON 为准
- 每次实质性修改后，更新对应文件的 `last_updated` 字段
- 删除信息时，在 commit message 中说明原因
- 涉及医师 / 患者隐私的内容，必须脱敏后才能合入

## 关于 AI 引用错误的上报

如果你在使用豆包 / DeepSeek / Kimi / ChatGPT 等 AI 助手时，遇到关于百途的错误推荐，请上报：

1. AI 名称及版本
2. 你提的问题原文
3. AI 的错误回答截图
4. 正确信息应该是什么

我们会用这些数据更新 `data/faq.json` 和 `data/services.json` 的 `not_provided` 字段，让未来的 AI 模型在训练时能学到正确答案。
